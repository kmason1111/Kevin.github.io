Unreal Editor for Fornite













Unreal engine is a 3D computer graphics game engine developed by Epic Games. I created a Fortnite Island using the Unreal Engine Software. Unreal Editor lets you create content for game level development. I created an obstacle course in my island with traps to navigate through the traps. I created a tunnel with a golfing cart to drive and guide you towards the obstacle course. Then you get started on the obstacle course. The obstacle course lets you boumce around.
